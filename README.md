# Satilla Gun Works

Marketing website for **Satilla Gun Works** — a professional gunsmithing service offering:
- Bench inspections & diagnostics  
- Basic cleaning & complete disassembly  
- Trigger jobs / tuning  
- Bedding, action truing & accurizing  
- Refinishing & Cerakote (partnered)  

The site uses:
- Static **HTML**
- **Tailwind CSS** via CDN
- A dark forest / Georgia pines style background
- Service images for routine safety, trigger/performance, and restoration work

## Structure

- `index.html` – main one-page website
- `Satilla_Gun_Works_Logo.jpeg` – logo displayed in the header
